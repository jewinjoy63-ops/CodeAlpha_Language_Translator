# Neural Machine Translation Studio

An attractive, interactive, and highly responsive single-page web application built to fulfill **Task 1 (Language Translation Tool)** for the CodeAlpha AI Model Development Internship.

## 🚀 Project Overview
This project delivers a modern translation portal utilizing a public translation API network gateway. It translates text asynchronously across multiple language vectors directly from the client browser without requiring complex server architectures or private API subscription keys.

---

## ✨ Features
* **Glassmorphism UI:** Built with Tailwind CSS utilities and featuring dynamic glow transitions, responsive grid scaling, and interactive font-awesome iconography.
* **Smart Performance Debouncer:** Features a built-in `debounceTimer` (500ms delay) that dynamically intercepts text area changes to optimize asynchronous API network quotas.
* **Dynamic Language Swap:** Allows users to flip the input/output language configurations and text contents instantly via a rotating interaction button.
* **Text-to-Speech (TTS) Integration:** Uses the browser-native HTML5 SpeechSynthesis engine to read translated outputs aloud in the correct target language accent.
* **One-Click Clipboard Macro:** Copies translated results instantly with custom-designed visual toast notification indicators.

---

## 🛠️ Tech Stack
* **Frontend Core:** HTML5, CSS3 (Custom Glassmorphism Filters)
* **Styling Framework:** Tailwind CSS CDN Engine
* **Icon Engine:** FontAwesome Framework v6.4.0
* **API Integration:** Asynchronous JavaScript (Fetch API Gateway)
* **Audio Layer:** Web Speech API (SpeechSynthesisUtterance)
